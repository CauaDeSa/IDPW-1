Fórum

Perguntas Frequentes:


a. Qual é o significado das entidades &lt;, &gt; e &amp;?

b. Para que serve o atributo href da tag <a>?

c. O que é URL?

d. Para que serve o atributo download da tag <a>?































Respostas

Resposta a:

São caracteres especiais do HTML que servem para inserir entidades especificas, ou para indicar o começo de uma referência de caractere, como é o caso do &.


Voltar ao Início

Resposta b:

Serve para especificar a página destino do link.


Voltar ao Início

Resposta c:

O Uniform Resource Location (URL), é um endereço de rede no qual se encontra algum recurso informático, é o que podemos observar na barra de pesquisa enquanto navegamos pela internet, visto que cada local navegado possui um endereçamento.


Voltar ao Início

Resposta d:

O atributo Download tem por objetivo sobrepor o comportamento dos navegadores e forçar o download dos arquivos.
